# Tipos de usuarios

Partes de este tema pueden estar traducidos automáticamente.

TrueConf Server admite la flexible asignación de roles para usuarios y administradores que trabajan con nuestro sistema de videoconferencia. A continuación, le explicaremos las capacidades que ofrece cada uno de ellos.

# Roles de usuario

Todos los usuarios conectados a TrueConf Server se dividen en las siguientes categorías:

  • Usuario – cuenta registrada en TrueConf Server. Cada usuario puede autenticarse a través de su cuenta utilizando cualquiera de los siguientes métodos:

    • en la aplicación cliente (para Windows/macOS/Linux, para Android, para iOS/iPadOS, e incluso para Android TV);

    • en área personal;

    • en terminales TrueConf Group o TrueConf Videobar;

    • en el terminal de software TrueConf Room;

    • con dispositivos SIP/H.323 que admiten la función de registro en Gatekeeper o PBX, como el altavoz de conferencia Phoenix Spider o el terminal Polycom HDX.

Si el usuario está registrado en el servidor pero no ha iniciado sesión, no se contabilizará dentro del recuento de usuarios en línea, cuyo límite se establece en la licencia.

Solo los usuarios autorizados en las aplicaciones para clientes o en sus áreas personales pueden organizar reuniones.

  • Invitado – es un usuario que participa en una videoconferencia de TrueConf sin autenticación. Participar en la conferencia como invitado solo es posible en una conferencia pública (webinar) mediante un enlace o después de registrarse previamente. En la conferencia, al invitado también le están disponibles los roles de moderador y presentador, los cuales se detallan a continuación.

Al crear una reunión pública, es posible restringir los derechos de los invitados, prohibiéndoles enviar mensajes, audio y vídeo.

  • Dispositivos SIP, H.323 y RTSP – terminales SIP/H.323 participantes en la conferencia (que no están registrados en el servidor) y flujos RTSP (por ejemplo, para la transmisión de imágenes desde cámaras IP).

Para obtener más información sobre cómo se licencia la conexión a TrueConf Server para usuarios de cada tipo, consulte la sección "Licenciamiento de TrueConf Server".

# Identificador de usuario

Cada usuario del servidor de videoconferencia TrueConf Server (así como del servicio en línea TrueConf Online) dispone obligatoriamente de un TrueConf ID.

TrueConf ID es un identificador único de usuario de TrueConf que se utiliza para la autorización en las aplicaciones de clientes, así como para direccionar videollamadas y reuniones.

TrueConf ID, por lo general, tiene el formato <user_id>@<server>. Aquí: <user_id> es el nombre de usuario que se proporcionó durante el registro; <server> es el nombre del servidor TrueConf.

Ejemplos:

Usuario george del servicio en la nube TrueConf Online:

george@trueconf.com

Usuario con el nombre maria en el TrueConf Server corporativo con el nombre server.company.com:

maria@server.company.com

A pesar del uso del símbolo @, TrueConf ID no es una dirección de correo electrónico. Si envía un correo a TrueConf ID del usuario, no le llegará.

# Roles de los participantes de la conferencia

Durante la realización del evento, todos los participantes asumen uno de los siguientes roles:

  • propietario de la conferencia — la persona que la creó y tiene acceso completo para administrarla;

  • moderador — asignado por el propietario o por otro moderador y puede hacer lo mismo que el propietario, excepto descargar grabaciones y acceder a análisis;

  • operador: pertenece a un grupo especial de operadores creado por el administrador en TrueConf Server, recibe los derechos de moderador en cualquier conferencia a la que se conecta;

  • orador — cualquier participante que todos ven y oyen (está en la tribuna);

  • intérprete — un rol especial de participante en una conferencia con el modo de interpretación simultánea. No se muestra en el diseño del vídeo, y su único rol en la conferencia es traducir las presentaciones de los ponentes a los idiomas seleccionados. Más detalles en los enlaces: crear una conferencia con interpretación simultánea, trabajar con interpretación simultánea en la aplicación cliente.

Para obtener más información sobre cada uno de los roles de los participantes de la conferencia, consulte la [documentación de la aplicación cliente TrueConf para Windows, Linux, macOS]().

# Roles administrativos

En TrueConf Server hay dos tipos de administradores de acuerdo con los grupos de usuarios que se agregan automáticamente al sistema operativo durante la instalación del servidor:

  • TrueConf Server Admin – tiene acceso completo al panel de control de TrueConf Server y puede cambiar cualquier configuración del servidor.

  • TrueConf Server Security Admin – solo tiene acceso para ver informes y grabaciones de videoconferencias. TrueConf Server Security Admin no puede cambiar la configuración en el panel de control de TrueConf Server.

Más detalles sobre el control de acceso al panel de administración de TrueConf Server se explican a continuación en la sección de configuración inicial.
