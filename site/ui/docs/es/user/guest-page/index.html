# ¿Qué es una página de invitado?

Partes de este tema pueden estar traducidos automáticamente.

Página de invitado: es la página de inicio que ve el usuario cuando se conecta a TrueConf Server desde el navegador.

Si eres administrador de TrueConf Server, desde la página de invitados puedes ir a su panel de control.

El enlace a la página de invitado tiene el formato http[s]://<servidor>[:<puerto>]. Por ejemplo: https://192.168.0.1:443 o https://video.company.com.

# Qué hay en la página de invitados

En la página de invitados, usted puede:

  • entrar al área personal;

  • descargar aplicaciones cliente para varios sistemas operativos;

  • programar una reunión (autenticación requerida);

  • conectarse a la conferencia por su ID;

  • consultar la guía del usuario.

Puede cambiar el idioma de la página de invitados haciendo clic en el icono de la bandera en la esquina superior izquierda, junto al logotipo.

Para obtener los datos de contacto del administrador del servidor, haga clic en el enlace Contact your administrator. Se abrirá una ventana que contiene su nombre, teléfono y correo electrónico.
